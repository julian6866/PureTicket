How does Pure Ticket work

Pure Ticket is the mobile client app that is used to download, scan, validate & synchronize with digital products (like e-tickets, coupons) sold from online eCommerce system.

How to try/demoe Pure Ticket

Download Tickets

Tap "Enter Auth Code" and input Auth code. (Use 'SAMPLE' as demo Auth Code).

The app will automatically pull down the tickets on the server.

You should see the total number of tickets is now updated (top right corner, right below the download icon). Now the Mobile Phone (Pure Ticket App) is ready to scan ticket (QR Codes).

Scan Ticket

Tap the icon to start scanning ticket.

Use Sample QR Codes (for demo purpose)

To scan ticket, you will need some sample valid QR codes. We have prepared them for you.

To make testing easier, please use a separate desktop/laptop computer or phone/table to open the following page so that your mobile phone (Pure Ticket App) can scan the QR codes on the screen:

http://www.pureiconsulting.com/tt

You may try to scan any QR code, but they will not be recognized as valid ticket.

Upload Ticket

You may manually upload scanned ticket to server so that they will be marked as "used".

Enable Sync with Server

You may enable Sync with server so that this app will upload scanned tickets and download & refresh unscanned tickets with server given the frequency you select by sliding the Sync bar.

FAQ

Is Internet connection required to use Pure Ticket?

The Internet connection is required only when initially downloading the tickets from the server.

It then can work in offline mode.

If you have multiple mobile devices running Pure Ticket app for scanning/validating the same set of tickets (using the same Auth Code), you will need Internet connection to allow mobile devices to synchronize with server at the frequency you set if you wish the devices are all synchronized at the time when tickets are scanned.
